Comentarios 
Al inicio, esta tarea sono retadora, puesto que no teniamos idea sobre como funciona un encriptamiento de contrasenias,
sin embargo, basto investigar para darse cuenta de que java ofrece algunas facilidades para implementar el algoritmo a traves de
el uso de algunas librerias. Por lo que en resumen, no estuvo tan dificil pero tomo tiempo de investigacion.
El funcionamiento de la encriptacion en el programa es el siguiente:
1. El usuario ingresa su usuario y contrasenia
2. Se encripta la contrasenia
3. Accedemos al HashMap que almacena los usuarios para verificar si existe el usuario
4. En caso de que exista, se comparara la contrasenia recien encriptada con la contrasenia encriptada del CSV
5. Si son iguales, entramos al sistema, de otro modo tira un mensaje.
